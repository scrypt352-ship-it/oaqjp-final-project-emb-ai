# Final Project

## oaqjp-final-project-emb-ai

This repository is my **Final Project** submission. The project is an AI-based web application that detects emotions in text using the Watson NLP library.

## Project Overview
This application analyzes user-provided text and identifies emotions such as anger, disgust, fear, joy, and sadness. It also returns the dominant detected emotion and provides a simple Flask-based web interface for interacting with the model.

## Features
- Detects anger, disgust, fear, joy, and sadness
- Returns the dominant emotion
- Provides a Flask web interface
- Includes unit testing and static code analysis

## Files
- `EmotionDetection/emotion_detection.py` - emotion detection logic
- `EmotionDetection/__init__.py` - package initialization
- `server.py` - Flask web server
- `test_emotion_detection.py` - unit tests

## How to Run the Application
```bash
python server.py
```

## How to Run Tests
```bash
python test_emotion_detection.py
```
